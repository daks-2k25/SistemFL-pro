# SistemFL Pro v4.0 🚀

O **SistemFL Pro** é um sistema de gerenciamento de credenciais desenvolvido em Python com interface moderna. Ele oferece um fluxo completo de autenticação, permitindo que administradores gerenciem acessos de usuários de forma segura e intuitiva.

## ✨ Funcionalidades
- **Painel Administrativo:** Acesso restrito para gestão (Cadastro, Listagem e Remoção).
- **Segurança:** Senhas protegidas via criptografia **SHA-256**.
- **Interface Moderna:** UI otimizada com `CustomTkinter` em Dark Mode.
- **Banco de Dados Local:** Persistência automática em arquivo JSON.
- **Identificação Única:** Geração automática de ID aleatório para cada novo usuário.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **UI:** CustomTkinter
- **Segurança:** Hashlib (SHA-256)
- **Dados:** JSON (Persistência local)
